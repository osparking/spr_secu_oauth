프로젝트 개요
* spring security : username(email) 과 비밀번호(password) 로그인
* 소셜 로그인: 구글 oauth 2 앱
  
특색들은 다음과 같다.
* 언어: 자바
* 프레임웍: 스프링 부트
* DBMS: MariaDB
* 프론트 기술: React
* 기타
  * JPA
  * Maven // Gradle 아님.
  * 롬복
